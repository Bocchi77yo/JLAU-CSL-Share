# JLAU-CSL-Share
吉林农业大学本科生毕业论文参考文献格式 CSL 样式（基于 《吉林农业大学本科生毕业论文参考模板(2024)》），适用于 Zotero。  
打开URL，即可通过Zotero Connector扩展自动安装：[https://raw.githubusercontent.com/Bocchi77yo/JLAU-CSL-Share/refs/heads/main/吉林农业大学(本科)-Zotero-20251122.csl](https://raw.githubusercontent.com/Bocchi77yo/JLAU-CSL-Share/refs/heads/main/吉林农业大学(本科)-Zotero-20251122.csl)   
或者你可以下载csl文件之后，手动在Zotero中导入。具体步骤如下：  
打开Zotero，选择编辑/Edit-设置/Setting-引用，点击样式列表右下角的“+”号，选择已经下载到本地的样式即可。
